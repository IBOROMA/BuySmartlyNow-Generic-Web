A modern, fast price comparison and affiliate marketing platform designed to help users save money and earn rewards.

## Features

- **Price Comparison**: Compare prices from top retailers like Amazon, Jumia, and more in real-time.
- **Partner Program**: Earn commissions by referring products. Includes a dedicated user dashboard.
- **Smart Deals**: Curated deals across Gadgets, Travel, Food, and Delivery.
- **Security**: Mocked Cloudflare security badge and simulated cloaked affiliate links.

## Tech Stack

- **Frontend**: HTML5, Vanilla JavaScript
- **Styling**: Tailwind CSS (CDN) + Custom CSS
- **Data**: Mock JS Objects (No backend required)
- **Auth**: LocalStorage simulation

## Setup

1.  Clone the repository.
2.  Open `index.html` in your browser.
3.  Enjoy exploring the deals!
